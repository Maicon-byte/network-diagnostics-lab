# 🌐 Network Diagnostics Lab

Projeto prático focado em diagnóstico de redes, simulando cenários reais enfrentados por analistas de suporte técnico.

---

## 🎯 Objetivo

Desenvolver habilidades essenciais de troubleshooting de rede:

- Verificar conectividade entre máquinas
- Diagnosticar problemas de DNS
- Analisar rotas de rede
- Interpretar configurações de rede

---

## 🖥️ Ambiente

Laboratório criado utilizando máquinas virtuais:

- Windows
- Linux (Ubuntu)
- VirtualBox

---

## 🛠️ Ferramentas utilizadas

```bash
ping
tracert / traceroute
nslookup
netstat
ipconfig / ifconfig
