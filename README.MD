# 🌐 Network Diagnostics Lab

Projeto prático focado em diagnóstico de redes, simulando situações reais enfrentadas por analistas de suporte técnico (N1).

---

## 🎯 Objetivo

Desenvolver habilidades essenciais de troubleshooting de rede na prática, incluindo:

- Verificação de conectividade entre máquinas
- Diagnóstico de falhas de DNS
- Análise de rotas de rede
- Identificação de portas abertas
- Interpretação de configurações de rede

---

## 🧩 Cenário real simulado

Este laboratório reproduz problemas comuns do dia a dia de suporte técnico, como:

- Falha de acesso à internet
- Erro na resolução de nomes (DNS)
- Configuração incorreta de gateway
- Falhas de comunicação entre máquinas

Todos os testes são realizados em um ambiente controlado utilizando máquinas virtuais.

---

## 🖥️ Ambiente

- 1 máquina virtual Windows  
- 1 máquina virtual Linux (Ubuntu)  
- VirtualBox para virtualização  
- Rede configurada para permitir comunicação entre as VMs  

---
